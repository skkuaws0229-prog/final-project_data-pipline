# LIHC v2 (간암) 앙상블 구성 지시서

## 1. 목적

간암(LIHC) v2 재학습 결과(Phase 2A/2B/2C)를 기반으로, **일반화 성능(GroupCV/ScaffoldCV ρ)**과 **과적합 억제(gap 최소화)**를 동시에 달성하는 앙상블을 구성하여 약물 추천에 활용한다.

---

## 2. LIHC v2 데이터 특성

### 2.1 v1 → v2 주요 변화

| 항목 | v1 | v2 |
|------|-----|-----|
| CatBoost 2B gen_mean | 0.5038 | **0.5039** (거의 동일) |
| ResidualStyle 2B 보정 점수 | — (미선택) | **0.3603** (전체 1위, 신규 투입) |
| 2x512 2C gen_mean | — (미선택) | **0.4818** (DL 1위, 신규 투입) |
| 앙상블 메인 1 | 1024_512_256 2B | **ResidualStyle 2B** |

→ v2에서 ResidualStyle 2B와 2x512 2C의 수치가 특히 개선되어, v1 지시서 대비 모델 구성이 변경됨.

### 2.2 LIHC v2 고유 특징

**① DL은 대부분 2C에서 groupcv ρ가 최고**
7개 DL 중 5개가 2C에서 groupcv ρ 최고. 다만 gap도 2C에서 다소 상승하는 모델이 있어 트레이드오프 존재.

**② ML은 2A/2B가 최적, 2C에서 하락**
CatBoost 2B가 최고(gcv ρ=0.5401), XGBoost 2A가 최고(0.5135), LightGBM 2A가 최고(0.4981). ML에서 2C는 모두 하락하므로 ML은 2A/2B를 사용.

**③ ML과 DL의 gap 격차가 큼**
ML 평균 gap ≈ 0.48, DL 평균 gap ≈ 0.27. ML이 DL 대비 약 1.8배 과적합. gen_mean에서는 ML(CatBoost)이 1위이나, 보정 점수 Top 15는 전부 DL.

**④ ResidualStyle 2B의 독보적 안정성**
scaffoldcv gap=0.2337로 전체 최저급이면서, groupcv ρ=0.5219, scaffoldcv ρ=0.4267로 성능도 DL 상위권. 보정 점수 전체 1위(0.3603).

### 2.3 Phase별 최적 모델 분포

| 모델 | 최적 Phase | groupcv ρ | 근거 |
|------|-----------|-----------|------|
| CatBoost | **2B** | 0.5401 | 2A≈2B > 2C |
| XGBoost | **2A** | 0.5135 | 2A > 2B > 2C |
| ResidualStyle | **2B** | 0.5219 | 2B > 2A > 2C |
| 2x512 | **2C** | 0.5377 | 2C > 2A > 2B |
| 3x512 | **2C** | 0.5318 | 2C > 2B > 2A (단, 2A gap 최저) |
| 1024_512_256 | **2C** | 0.5084 | 2C > 2B > 2A (단, 2A/2B gap 최저) |

→ ML은 2A/2B, DL은 2C 경향이 뚜렷하나, gap을 고려하면 DL도 2A/2B가 유리한 경우 존재.

## 3. 최종 앙상블 구성

| 역할 | 모델 | Phase | 가중치 | 선정 근거 |
|------|------|-------|--------|----------|
| 메인 1 | DL_MLP_ResidualStyle | 2B | **0.25** | 보정 점수 1위 (0.3603), scv_gap=0.2337 전체 최저 |
| 메인 2 | CatBoost | 2B | **0.25** | gen_mean 전체 1위 (0.5039), groupcv ρ=0.5401 |
| 보조 1 | DL_MLP_2x512 | 2C | **0.17** | DL groupcv ρ=0.5377 최고, gen_mean=0.4818 DL 1위 |
| 보조 2 | DL_MLP_3x512 | 2A | **0.15** | 보정 점수 2위 (0.3564), avg_gap=0.2160 최저급 |
| ML 보조 | XGBoost | 2A | **0.12** | gen_mean=0.4809 ML 2위, groupcv ρ=0.5135 |
| 안정성 보조 | DL_MLP_1024_512_256 | 2B | **0.06** | avg_gap=0.2777 양호, 보정 점수 6위 (0.3281) |

### 가중 평균 수식

```
최종 예측 = 0.25 × ResidualStyle_2B(x)
           + 0.25 × CatBoost_2B(x)
           + 0.17 × MLP_2x512_2C(x)
           + 0.15 × MLP_3x512_2A(x)
           + 0.12 × XGBoost_2A(x)
           + 0.06 × MLP_1024_512_256_2B(x)
```

### 비율 요약

- **DL 63%** (ResidualStyle 0.25 + 2x512 0.17 + 3x512 0.15 + 1024_512_256 0.06)
- **ML 37%** (CatBoost 0.25 + XGBoost 0.12)
- 모델 6개, 아키텍처 5종 (Residual MLP, Ordered GBDT, 2-layer MLP, 3-layer MLP, Histogram GBDT)
- Phase 3종 (2A, 2B, 2C) 모두 포함

## 4. 설계 원칙

### 4.1 DL 63% / ML 37%의 근거

| 구분 | 모델 | groupcv gap | scaffoldcv gap | 평균 gap |
|------|------|-----------|--------------|---------|
| DL | ResidualStyle 2B | 0.2470 | **0.2337** | **0.2404** |
| DL | 2x512 2C | 0.2900 | 0.3554 | 0.3227 |
| DL | 3x512 2A | 0.2308 | 0.2012 | **0.2160** |
| DL | 1024_512_256 2B | 0.2261 | 0.3294 | 0.2777 |
| ML | CatBoost 2B | 0.4273 | 0.4991 | **0.4632** |
| ML | XGBoost 2A | 0.4826 | 0.5476 | **0.5151** |

→ DL 평균 gap ≈ 0.26, ML 평균 gap ≈ 0.49. ML이 DL 대비 약 1.9배 과적합.
→ ML이 gen_mean에서 앞서더라도 gap 격차를 고려하면 DL 중심이 안전.

### 4.2 Phase 선택 기준

- **ML → 2A/2B**: CatBoost 2B (gcv ρ 최고), XGBoost 2A (gcv ρ 최고). ML에서 2C는 모두 하락.
- **DL 보조 → 2C 또는 2A**: 2x512는 2C에서 gcv ρ 최고. 3x512는 2C에서 gcv ρ 최고이나 2A의 gap이 훨씬 낮아(0.22 vs 0.31) 안정성 우선으로 2A 선택.
- **DL 메인 → 2B**: ResidualStyle 2B가 성능+안정성 모두 최고. 1024_512_256도 2B에서 gap이 가장 낮음.

### 4.3 모델 제외 사유

| 제외 모델 | 사유 |
|----------|------|
| GAT | groupcv ρ = −0.003~0.176, 학습 실패 |
| GraphSAGE | groupcv ρ = 0.08~0.24, 성능 극히 낮음 |
| SVR_RBF | gen_mean=0.34, underfitting |
| ExtraTrees | groupcv gap=0.53~0.54, 과적합 심각 |
| RandomForest | groupcv gap=0.49~0.51, 과적합 심각 |
| LightGBM | CatBoost·XGBoost 대비 gen_mean·gap 모두 열위 |
| SELU_3x256 | scaffoldcv ρ=0.35~0.39 불안정, gen_mean 하위권 |
| 2x1024 | 2x512 대비 gen_mean 열위, 유사 아키텍처 |
| WideNarrow | gen_mean=0.4710(2C)이나 상위 DL로 충분, 슬롯 부족 |

### 4.4 다양성 확보 구조

| 모델 | Family | 아키텍처 유형 | Phase | 피처셋 |
|------|--------|-------------|-------|--------|
| ResidualStyle | DL | Residual 연결 MLP | 2B | numeric + SMILES |
| CatBoost | ML | Ordered GBDT | 2B | numeric + SMILES |
| 2x512 | DL | 2-layer 균등 MLP | 2C | numeric + SMILES + context |
| 3x512 | DL | 3-layer 균등 MLP | 2A | numeric only |
| XGBoost | ML | Histogram GBDT | 2A | numeric only |
| 1024_512_256 | DL | 점감형 3-layer MLP | 2B | numeric + SMILES |

→ 아키텍처 5종 (Residual MLP, Ordered GBDT, 2-layer MLP, 3-layer MLP, Histogram GBDT)
→ Phase 3종 (2A, 2B, 2C) 모두 포함
→ 피처셋 3종 (numeric only, numeric+SMILES, numeric+SMILES+context)

## 5. 앙상블 방법

### 5.1 기본 방식: 가중 평균 (Weighted Average)

위 가중치로 각 모델의 예측값을 가중 평균하여 최종 예측 산출.

### 5.2 고급 방식: Stacking (선택 사항)

가중 평균으로 충분한 성능이 나오지 않을 경우 적용.

- **1층**: 선택된 6개 모델의 예측값 출력
- **2층 메타 러너**: Ridge Regression 또는 ElasticNet
- **핵심**: 메타 러너 학습 시 반드시 **GroupCV 또는 ScaffoldCV 분할** 사용 (CV5 사용 금지)

## 6. 약물 추천 적용

### 6.1 추천 기준

1. 앙상블 예측값 상위 N개 약물 선정
2. 예측 불확실성 필터링: 6개 모델 간 예측값 분산이 낮은 약물 우선 추천

### 6.2 신뢰도 등급

| 등급 | 조건 | 해석 |
|------|------|------|
| A (높음) | 6개 모델 모두 상위 예측 + 분산 낮음 | 강력 추천 |
| B (보통) | 다수 모델 상위 예측 + 분산 중간 | 조건부 추천 |
| C (낮음) | 모델 간 예측 불일치 + 분산 높음 | 추가 검증 필요 |

## 7. 검증 계획

1. **앙상블 성능 측정**: GroupCV ρ, ScaffoldCV ρ로 개별 모델 대비 개선 확인
2. **가중치 미세 조정**: Optuna 등으로 GroupCV 기준 최적 가중치 탐색
3. **Ablation 테스트**: 모델 1개씩 제거하며 각 모델의 기여도 확인
4. **1024_512_256 제거 비교**: 0.06 가중치 모델 제거 시 5개 구성과 성능 비교
5. **개별 모델 대비 개선 확인**: 앙상블 ρ > 최고 단일 모델 ρ (CatBoost 2B groupcv ρ=0.5401) 달성 여부
