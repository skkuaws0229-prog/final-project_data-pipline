# LIHC (간암) 앙상블 구성 지시서

## 1. 목적

간암(LIHC) 모델 학습 결과(Phase 2A/2B/2C)를 기반으로, **일반화 성능(GroupCV/ScaffoldCV ρ)**과 **과적합 억제(gap 최소화)**를 동시에 달성하는 앙상블을 구성하여 약물 추천에 활용한다.

---

## 2. LIHC 데이터 특성

### 2.1 LIHC 고유 특징

**① ML이 Phase별로 다른 수치를 보임**
HNSC와 달리 ML도 Phase별로 다른 학습 결과를 보이며, 대부분 2A≈2B > 2C 패턴. CatBoost groupcv ρ: 2A=0.5394, 2B=0.5401, 2C=0.5148로 2C에서 하락.

**② CatBoost가 gen_mean 1위이지만, 보정 점수에서는 DL이 압도적**
CatBoost 2B의 gen_mean=0.5038로 전체 1위이나 평균 gap=0.46. DL_MLP_1024_512_256 2B는 gen_mean=0.4789이지만 보정 점수 0.3782로 전체 1위 (평균 gap=0.21).

**③ ML과 DL의 gap 격차가 매우 큼**
ML 평균 gap ≈ 0.48 vs DL 평균 gap ≈ 0.26. ML이 DL 대비 약 2배 과적합. gen_mean에서 ML이 약간 앞서더라도 실제 일반화 신뢰도는 DL이 훨씬 높음.

**④ Phase 2C의 context 피처는 대체로 해로움**
ML과 DL 모두 2C에서 성능이 떨어지거나 정체하는 경향. PDAC와 유사한 패턴으로, 2C는 제한적으로만 사용.

**⑤ DL 모델별 최적 Phase가 다름**
1024_512_256, 2x1024, SELU, ResidualStyle → 2B 최적. 2x512, 3x512, WideNarrow → 2C 최적. 모델별로 최적 Phase를 개별 선택함.

### 2.2 보정 점수 상위 모델

보정 점수 = gen_mean × (1 − 평균 gap)으로, 성능과 과적합을 동시에 반영함.

| 순위 | Phase | Family | 모델 | Gen Mean | 평균 gap | 보정 점수 |
|------|-------|--------|------|----------|---------|----------|
| 1 | 2B | DL | 1024_512_256 | 0.4789 | 0.2102 | **0.3782** |
| 2 | 2C | DL | 3x512 | 0.4695 | 0.2725 | **0.3415** |
| 3 | 2A | DL | 2x512 | 0.4307 | 0.2240 | 0.3342 |
| 4 | 2A | DL | SELU_3x256 | 0.4431 | 0.2476 | 0.3334 |
| 5 | 2A | DL | 2x1024 | 0.4673 | 0.2877 | 0.3329 |
| 6 | 2B | DL | SELU_3x256 | 0.4569 | 0.2783 | 0.3297 |
| — | 2B | ML | CatBoost | 0.5038 | 0.4632 | 0.2705 |
| — | 2A | ML | XGBoost | 0.4809 | 0.5151 | 0.2332 |

→ 보정 점수 상위 6개가 모두 DL. ML은 gen_mean이 높지만 gap이 커서 보정 점수에서 크게 밀림.

## 3. 최종 앙상블 구성

| 역할 | 모델 | Phase | 가중치 | 선정 근거 |
|------|------|-------|--------|----------|
| 메인 1 | DL_MLP_1024_512_256 | 2B | **0.25** | 보정 점수 1위 (0.3782), gap=0.21 최저 |
| 메인 2 | CatBoost | 2B | **0.22** | gen_mean 전체 1위 (0.5038), groupcv ρ=0.5401 |
| 보조 1 | DL_MLP_SELU_3x256 | 2B | **0.18** | DL groupcv ρ 2위 (0.5361), gap=0.28 안정 |
| 보조 2 | DL_MLP_3x512 | 2C | **0.15** | 보정 점수 2위 (0.3415), gap=0.27 |
| ML 보조 | XGBoost | 2A | **0.12** | groupcv ρ=0.5135, ML 다양성 확보 |
| DL 보조 | DL_MLP_WideNarrow | 2C | **0.08** | gen_mean=0.4700, 아키텍처 다양성 |

### 가중 평균 수식

```
최종 예측 = 0.25 × MLP_1024_512_256_2B(x)
           + 0.22 × CatBoost_2B(x)
           + 0.18 × MLP_SELU_3x256_2B(x)
           + 0.15 × MLP_3x512_2C(x)
           + 0.12 × XGBoost_2A(x)
           + 0.08 × MLP_WideNarrow_2C(x)
```

### 비율 요약

- **DL 66%** (0.25 + 0.18 + 0.15 + 0.08)
- **ML 34%** (0.22 + 0.12)
- 모델 6개, 아키텍처 6종 모두 다름

## 4. 설계 원칙

### 4.1 DL 66% / ML 34%의 근거

| 구분 | 모델 | groupcv gap | scaffoldcv gap | 평균 gap |
|------|------|-----------|--------------|---------|
| DL | 1024_512_256 2B | 0.2588 | 0.1616 | **0.2102** |
| DL | SELU_3x256 2B | 0.2842 | 0.2724 | **0.2783** |
| DL | 3x512 2C | 0.2137 | 0.3313 | **0.2725** |
| DL | WideNarrow 2C | 0.2314 | 0.3867 | **0.3090** |
| ML | CatBoost 2B | 0.4273 | 0.4991 | **0.4632** |
| ML | XGBoost 2A | 0.4826 | 0.5476 | **0.5151** |

→ ML 평균 gap ≈ 0.49, DL 평균 gap ≈ 0.27. ML이 DL 대비 약 2배 과적합.
→ ML이 gen_mean에서 약간 앞서더라도, gap 격차를 고려하면 DL 중심이 더 안전.

### 4.2 Phase 선택 기준

- **DL 메인 → 2B 선택**: 1024_512_256, SELU 모두 2B에서 최적 성능.
- **DL 보조 → 2C 선택**: 3x512, WideNarrow은 2C에서 최적. Phase 다양성 확보.
- **ML → 각각 최적 Phase**: CatBoost는 2B (gen_mean 최고), XGBoost는 2A (groupcv ρ 최고).
- **2C 비중 제한**: 전체의 23% (0.15 + 0.08)만 2C. context 피처 해로움 반영.

### 4.3 모델 제외 사유

| 제외 모델 | 사유 |
|----------|------|
| GAT | 음수 ρ 빈발, 학습 실패 |
| GraphSAGE | gen_mean=0.11~0.15, 성능 극히 낮음 |
| SVR_RBF | gen_mean=0.34, underfitting |
| ExtraTrees | groupcv gap=0.53~0.55, 과적합 심각 |
| RandomForest | groupcv gap=0.49~0.51, 과적합 심각 |
| LightGBM | CatBoost·XGBoost 대비 gen_mean·gap 모두 열위 |
| DL_MLP_2x1024 | 1024_512_256과 유사 아키텍처, 슬롯 부족 |
| DL_MLP_2x512 | gen_mean 하위권 |
| DL_MLP_ResidualStyle | 선택된 모델들과 동급이나 슬롯 부족 |

### 4.4 다양성 확보 구조

| 모델 | Family | 아키텍처 유형 | Phase | 피처셋 |
|------|--------|-------------|-------|--------|
| 1024_512_256 | DL | 점감형 3-layer MLP | 2B | numeric + SMILES |
| CatBoost | ML | Ordered GBDT | 2B | numeric + SMILES |
| SELU_3x256 | DL | SELU 활성화 3-layer MLP | 2B | numeric + SMILES |
| 3x512 | DL | 균등 3-layer MLP | 2C | numeric + SMILES + context |
| XGBoost | ML | Histogram GBDT | 2A | numeric only |
| WideNarrow | DL | Wide→Narrow 점감 MLP | 2C | numeric + SMILES + context |

→ 아키텍처 6종 모두 다름
→ Phase 3종 (2A, 2B, 2C) 모두 포함
→ 피처셋 3종 (numeric only, numeric+SMILES, numeric+SMILES+context)

## 5. 앙상블 방법

### 5.1 기본 방식: 가중 평균 (Weighted Average)

위 가중치로 각 모델의 예측값을 가중 평균하여 최종 예측 산출.

### 5.2 고급 방식: Stacking (선택 사항)

가중 평균으로 충분한 성능이 나오지 않을 경우 적용.

- **1층**: 선택된 6개 모델의 예측값 출력
- **2층 메타 러너**: Ridge Regression 또는 ElasticNet
- **핵심**: 메타 러너 학습 시 반드시 **GroupCV 또는 ScaffoldCV 분할** 사용 (CV5 사용 금지)

## 6. 약물 추천 적용

### 6.1 추천 기준

1. 앙상블 예측값 상위 N개 약물 선정
2. 예측 불확실성 필터링: 6개 모델 간 예측값 분산이 낮은 약물 우선 추천

### 6.2 신뢰도 등급

| 등급 | 조건 | 해석 |
|------|------|------|
| A (높음) | 6개 모델 모두 상위 예측 + 분산 낮음 | 강력 추천 |
| B (보통) | 다수 모델 상위 예측 + 분산 중간 | 조건부 추천 |
| C (낮음) | 모델 간 예측 불일치 + 분산 높음 | 추가 검증 필요 |

## 7. 검증 계획

1. **앙상블 성능 측정**: GroupCV ρ, ScaffoldCV ρ로 개별 모델 대비 개선 확인
2. **가중치 미세 조정**: Optuna 등으로 GroupCV 기준 최적 가중치 탐색
3. **Ablation 테스트**: 모델 1개씩 제거하며 각 모델의 기여도 확인
4. **개별 모델 대비 개선 확인**: 앙상블 ρ > 최고 단일 모델 ρ (CatBoost 2B groupcv ρ=0.5401) 달성 여부

## 8. 4개 암종 앙상블 비교

### 8.1 구성 비교

| 항목 | PDAC (췌장암) | BRCA (유방암) | HNSC (두경부암) | LIHC (간암) |
|------|-------------|-------------|----------------|------------|
| **DL:ML 비율** | 75:25 | 55:45 | 78:22 | **66:34** |
| **모델 수** | 5개 | 5개 | 6개 | **6개** |
| **Phase 전략** | 2C 제외 | 2C 중심 | 2A/2B/2C 혼용 | **2B 중심, 2C 제한** |
| **1위 모델** | MLP_3x512 (DL) | CatBoost (ML) | SELU_3x256 (DL) | **CatBoost (ML) gen_mean / 1024_512_256 (DL) 보정** |
| **ML Phase** | 2A | 2C | Phase 무관 | **2A/2B 혼용** |
| **2C 비중** | 0% | ~100% | ~33% | **23%** |

### 8.2 전략 차이의 근거

**PDAC — DL 75%, 2C 전면 제외**
Phase 2C에서 cv5만 상승하고 groupcv/scaffoldcv가 하락. Context 피처가 과적합을 유발. DL이 gap 면에서 ML보다 우수.

**BRCA — ML 45%, 2C 중심**
Phase 2C에서 groupcv ρ 상승 + gap 감소 동시 달성. Context 피처가 실제 유효. CatBoost 2C가 전체 1위.

**HNSC — DL 78%, Phase 혼용**
ML 수치가 Phase 간 동일. SELU_3x256이 전 Phase 압도적 1위. DL과 ML gap 격차 1.5배.

**LIHC — DL 66%, 2B 중심 + 2C 제한**
ML이 gen_mean에서 앞서지만 gap이 DL의 2배. 2C는 대부분 모델에서 하락하나 일부 DL(3x512, WideNarrow)에서는 유효. 모델별 최적 Phase가 달라 개별 선택 필요.

### 8.3 암종별 핵심 교훈

| 교훈 | 해당 암종 |
|------|----------|
| gen_mean 1위 ≠ 앙상블 메인이어야 하는 것은 아님 (gap 고려 필요) | LIHC |
| 같은 모델 Phase 중복보다 아키텍처 다양성이 우선 | LIHC, BRCA |
| Context 피처 효과는 암종마다 다름 (해로움~유효 스펙트럼) | 전체 |
| ML Phase가 동일할 수 있으며, 파이프라인 확인 필요 | HNSC |
| DL:ML 비율은 gap 격차에 비례해야 함 | 전체 |
| 보정 점수 = gen_mean × (1−avg_gap)이 단순 gen_mean보다 신뢰도 높음 | LIHC |
